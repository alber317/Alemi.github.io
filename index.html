Dedicatoria

Hola. Cada vez que pienso en vos me acuerdo de cuando nos decíamos que vos eras Gojo y yo era DALL·E. Por eso hice esto: una página para guardar nuestros chistes, fotos y las cosas que nos representan. Gracias por ser parte de esto.

P.D. Si querés que cambie algo, copiá y pegá el texto que querés cambiar y yo lo edito.

Abrir en el navegador
Escaneá con la cámara o tocá el botón para abrir esta página.
Cómo usar este archivo
  1. Subí este archivo al repo de GitHub que quieras publicar (o creá un repo nuevo llamado tuusuario.github.io para publicar a la raíz de tu usuario).
  2. Si generás un QR, subilo al mismo repo y reemplazá el atributo src de #qrimg por ./qr.png (o la ruta que uses).
  3. Para publicar: en GitHub -> Settings -> Pages (o en "Code and automation" -> Pages) seleccioná la rama main y la carpeta / (root). GitHub tardará unos segundos en publicar y te dará la URL https://tuusuario.github.io.
  4. También podés editar directamente en GitHub: abrí el archivo, clic en ✏️ (edit), pegá tu texto y 'Commit changes'.

Si querés, ahora mismo pegá aquí el texto definitivo que querés que aparezca y las URLs de 1–4 imágenes (por ejemplo, imágenes de Jujutsu Kaisen/Gojo) y lo edito ahora mismo y te devuelvo el archivo listo para subir.
